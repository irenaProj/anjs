Rendering Engines

Major CSS/HTML rendering engines are:

  • Gecko - a web browser engine used in many applications developed by Mozilla. Gecko is written in C++ and is cross-platform
  • WebKit - a layout engine software component for rendering web pages in web browsers. It powers Apple's Safari web browser and a fork of the project is used in Google's Chrome web browser.
  • Trident - a layout engine for the Microsoft Internet Explorer
  • Presto - no longer developed; Opera and Opera Mobile use Blink now
  • Blink - a web browser engine developed by Google with contributions from Opera, Intel, Samsung and others. It was first announced in April 2013. It is a fork of the WebCore component of WebKit and is used in Chrome.
